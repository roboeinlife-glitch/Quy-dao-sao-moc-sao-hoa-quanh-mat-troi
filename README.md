# Quy-dao-sao-moc-sao-hoa-quanh-mat-troi

Đây là một chương trình C++ SFMl được viết bởi Grok AI

Lấy cảm hứng từ bài đăng của [Psikoloji Rehberi](https://x.com/pskolojirehberi/status/1454011906742755337)

Tôi đã tạo ra video ở đây: [Jupiter and Mars orbit the Sun with file](https://www.youtube.com/shorts/eXoc3ffGJwo)

Do tôi chưa biết cách đưa file gif vào code nên ảnh động mặt trời trong video là nhờ vào ứng dụng tạo video để ghép hình ảnh vào.

Cách làm video như sau:

Sử dụng các file png trong code, sau đó chạy chương trình và ghi hình lại

Tải một file gif mặt trời về sau đó dùng ứng dụng cắt hình gif trực tuyến để làm tròn mặt trời (do file gif thường có viền vuông bên ngoài)

Sau khi đã có ảnh gif tròn, bạn chỉ việc kéo vào vừa khớp với ảnh tĩnh và hoàn thành việc tạo video là xong
